How This Works

When you click "Subscribe", you'll be redirected to Stripe's secure payment page. This is the recommended approach because:

  • Stripe handles all payment processing and security
  • Your card details go directly to Stripe, never to our servers
  • PCI compliant by design - we never see your card number
  • Works on all devices with optimized mobile experience
  • Supports multiple payment methods (cards, Apple Pay, Google Pay)

How to get your Price IDs:

  1. Go to Stripe Dashboard ‚Üí Products
  2. Create your products (Pro, Enterprise)
  3. Click on the product and copy the Price ID (starts with price_)
  4. Update the IDs in both this HTML file and services/payment_config.py

Frontend Code (super simple):

async function redirectToCheckout(priceId, plan) { const response = await fetch('/api/payment/create-checkout-session', { method: 'POST', headers: {'Content-Type': 'application/json'}, body: JSON.stringify({ price_id: priceId, mode: 'subscription', plan: plan }) }); const {checkout_url} = await response.json(); window.location.href = checkout_url; // Redirect to Stripe! }
